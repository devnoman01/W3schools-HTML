HTML links are hyperlink

If we click here then google home page will come.

HTML link syntax:

        < a href="url"> link text < /a >

        < a href="http://google.com"> Google < /a >

Target (_blank,_self,_parent,_top) attribute specifies where to open the linked document

Open in a new window or tab
Open in current/same window or tab
Open in the parent frame
Open in the full body of the window

Absolute URL

        < a href="https://www.google.com/"> Google < /a >

Relative URL

        < a href="/files/default.asp"> Download < /a >
