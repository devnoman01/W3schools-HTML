A web browser must know the character set used in the page to display an HTML page correctly.

That's why charset attribute is used in HTML

charset attribute is specified in < meta > tag

        < meta charset="UTF-8" >

Some Common form of charset:

ASCII - American Standard Code for Information Interchange

ISO-8859-1 was the default character set for HTML 4.

ANSI - American National Standards Institute (Windows-1252) was the original Windows character set

HTML5 specification encourages web developers to use the UTF-8 (Unicode Transformation Format - 8 )character set
