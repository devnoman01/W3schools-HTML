Use < code > tag for defining user input and computer code.

        < code >
            x = 5;
            y = 6;
            z = x + y;
        < /code >

Output:

x = 5; y = 6; z = x + y;

The < kbd > element is used to define keyboard input. Check the sample below:

        < p >Save the document by pressing < kbd > Ctrl + S< /kbd >< /p >

Output

Save the document by pressing Ctrl + S
