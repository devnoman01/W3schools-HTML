Colors can be specified using RGB, HEX, HSL as shown below:

HEX values - example: #008000

RGB values - example: rgb(0, 128, 0)

RGBA values - example: rgb(0, 128, 0, 0.5) - Here "a" means alpha = value of transparency from 0.0 to 1.0

HSL values - example: hsl(120, 100%, 25%)

HSLA values - example: hsl(120, 100%, 25%, 0.5) - Here "a" means alpha = value of transparency from 0.0 to 1.0
