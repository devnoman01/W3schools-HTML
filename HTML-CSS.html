CSS - Cascading Style Sheets is used to format the layout of a webpage

CSS can be added to HTML document in 3 way:

Inline - attribute used inside the tag

< h1 style="color:forestgreen;">A Blue Heading< /h1>

Internal - using style attribute inside head tag

        < head >
            < style >
                h4{color: orangered;}
            < /style >
        < /head >
        < body >
            < h4 > A Orangered Heading < /h4 >
        < /body >

External - HTML file is linked with a file having .css extension

Link tag using format:- (applied when stylesheet located in same folder)
< link rel="stylesheet" href="styles.css" >
External style sheets can be referenced with a full URL
< link rel="stylesheet" href="https://www.w3schools.com/html/styles.css" >
